### Table of contents

### Install

#### Clone this repo

```
git clone https://github.com/warwicklol14/electron-react-ts.git
```

#### Install dependencies

```
npm install
```

### Usage

#### Run the app

```
npm start
```

#### Build the app for distribution(release)

```
npm run release-build
```
